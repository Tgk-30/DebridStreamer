Self-host

Run the server where it makes sense.

Use an always-on desktop, NAS, VPS, Raspberry Pi, or private home server. The server owns profile isolation, encrypted credentials, stream sessions, bandwidth visibility, and the debrid-facing IP.

The desktop app can also start Server Mode from Settings, then open the hosted PWA for phones, tablets, and other computers with a copyable setup URL and QR code.

Owners can send invite links so each person creates their own profile, password, history, and optional credential overrides. Profiles can also keep stream results cached-only or cap visible quality/file size before playback.

Updates

Signed OTA updates for the pre-bundled app.

The desktop app checks GitHub Releases for a signed latest.json. When a new version is available, the in-app updater can download, install, and relaunch into the new build. Auto-checks can be controlled from Settings.
